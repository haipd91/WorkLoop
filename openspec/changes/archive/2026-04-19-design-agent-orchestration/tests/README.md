# Tests placeholder

Change này sửa kiến trúc orchestration của `/design`, không phải output feature
runtime cụ thể.

Artifact `tests` được giữ ở mức placeholder để change này sẵn sàng cho bước
implementation. Nếu cần, bước `/tdd` sau đó có thể tạo scenario files mô tả:

- case dùng core specialist agents
- case dùng optional agents
- case conflict resolution giữa nhiều agent outputs
